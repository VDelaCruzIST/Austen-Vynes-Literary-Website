# Austen Vynes Literary Website

## Overview
This repository contains a literary-themed website developed as part of my coursework while pursuing a Bachelor of Business Administration in Information Systems and Technology (IST).

The website focuses on the life and works of Jane Austen and demonstrates modern web development concepts including JavaScript programming, random quote generation, custom font integration, responsive layouts, and dynamic webpage content.

## Features
* Literary and author-focused website
* Random Jane Austen quote generator
* Dynamic content generation using JavaScript
* Custom web font integration
* Sidebar navigation system
* Biography and literary content presentation
* Styled quotations and visual effects
* Multi-column footer navigation

## Technologies Used
* HTML5
* CSS3
* JavaScript
* Custom Web Fonts
* DOM Manipulation

## Project Files
* `ja_vynes.html` – Main website page
* `ja_base.css` – Base stylesheet
* `ja_layout.css` – Layout and visual styling
* `ja_quote.js` – Random quote generation script
* `DobkinPlain.ttf` – Custom font file
* `DobkinPlain.woff` – Web font file
* Website image and branding assets

## Skills Demonstrated

* DOM manipulation
* Custom font implementation
